## 🔧 Быстрый запуск

### 1. Активация виртуального окружения
```bash
source venv/bin/activate
```

### 2. Запуск бота
```bash
python3 app.py
```

## 📝 Настройка

### Переменные окружения (.env файл)
```env
# Токен вашего бота (уже настроен)
BOT_TOKEN=ваш_токен_бота

# Название проекта
PROJECT_NAME=shop-bot

# ID администраторов (через запятую)
ADMINS=123456789,987654321

# Настройки webhook (опционально)
RAILWAY_PUBLIC_DOMAIN=
WEBHOOK_PATH=
```

## 🎯 Режимы работы

- **Polling** (по умолчанию) - для локальной разработки
- **Webhook** - для продакшена на Heroku/Railway

## 🐛 Устранение проблем

Если возникают ошибки:

1. **Проверьте виртуальное окружение**: `source venv/bin/activate`
2. **Установите зависимости**: `pip install -r requirements.txt`
3. **Проверьте токен**: убедитесь, что BOT_TOKEN установлен в .env
4. **Запустите тест**: `python3 test_structure.py` (если есть)
